# Smart Feed Engine V2.1 - Quick Reference

## 🏗️ NEW Architecture

```
SmartFeedDecisionEngine (Intelligence)
         ↓
  SmartFeedOutput (Complete Decision)
         ↓
  SmartFeedHelper.Mapper (Simple conversion)
         ↓
   FeedResult (UI Data)
         ↓
 SmartFeedDebugScreen (Renders only)
```

**Key: Logic in Engine. No logic in UI.** ✅

## 🚀 Architecture Upgrade (NEW)

### BEFORE ❌
Helper built explanation + confidence

### AFTER ✅
Engine builds everything (explanation, confidence, recommendations)

```dart
// Engine decides EVERYTHING
final output = SmartFeedDecisionEngine.buildSmartFeedOutput(
  finalFeed: 10.2,
  docFeed: 11.0,
  biomassFeed: 10.8,
  abw: 12.5,
  doc: 35,
  fcrFactor: 0.91,
  trayFactor: null,
  growthFactor: 1.0,
  samplingAgeDays: 3,
);

// output now contains:
// - output.explanation (generated by engine)
// - output.confidenceScore (calculated by engine)
// - output.recommendations (generated by engine)
// - output.source (determined by engine)
```

## 🔧 Core Engine Methods

### 1. buildSmartFeedOutput() - Orchestrator
```dart
final output = SmartFeedDecisionEngine.buildSmartFeedOutput(
  finalFeed: 10.2,
  docFeed: 11.0,
  biomassFeed: 10.8,
  abw: 12.5,
  doc: 35,
  fcrFactor: 0.91,
  trayFactor: null,
  growthFactor: 1.0,
  samplingAgeDays: 3,
);

// Returns: SmartFeedOutput (complete decision)
```

### 2. buildExplanation() - Natural Language
```dart
final explanation = SmartFeedDecisionEngine.buildExplanation(
  source: FeedSource.biomass,
  docFeed: 11.0,
  finalFeed: 10.2,
  fcrFactor: 0.91,
  trayFactor: null,
  growthFactor: 1.0,
);

// Returns:
// "• Biomass data detected from recent sampling
//  • FCR = 1.10 → Overfeeding risk detected
//  • Feed reduced by 7% for safety"
```

### 3. calculateConfidenceScore() - Data Quality
```dart
final confidence = SmartFeedDecisionEngine.calculateConfidenceScore(
  hasRecentSampling: true,
  samplingAgeDays: 3,      // Fresh = more points
  hasFcrData: true,
  hasTrayData: false,
  hasGrowthData: true,
  doc: 35,                 // Smart phase bonus
);

// Returns: 0.0 - 1.0
// 0.9+ = Very confident
// 0.7-0.9 = Confident
// 0.5-0.7 = Moderate
// < 0.5 = Low confidence
```

### 4. generateRecommendations() - Action Items
```dart
final recs = SmartFeedDecisionEngine.generateRecommendations(
  fcrFactor: 0.91,
  trayFactor: null,
  growthFactor: 1.0,
  samplingAgeDays: 3,
  confidenceScore: 0.82,
  source: FeedSource.biomass,
);

// Returns: List<String> of actionable items
// ["→ Slightly reduce feed for next 2 days",
//  "→ Monitor tray closely for overflow",
//  "📊 Plan sampling measurement within 2 days"]
```

### 5. determineFeedSource() - Smart Selection
```dart
final source = SmartFeedDecisionEngine.determineFeedSource(
  abw: 12.5,              // null or value?
  samplingAgeDays: 3,     // how old?
);

// Returns: FeedSource.biomass or FeedSource.doc
```

## 📊 SmartFeedOutput Structure

```dart
class SmartFeedOutput {
  // Calculation results
  double finalFeed;           // 10.2 kg
  FeedSource source;          // biomass or doc
  double docFeed;             // 11.0 kg (baseline)
  double? biomassFeed;        // 10.8 kg (optional)

  // Individual factors
  double? fcrFactor;          // 0.91 (overfeeding)
  double? trayFactor;         // null (not available)
  double? growthFactor;       // 1.0 (normal)
  int? samplingAgeDays;       // 3 (recent)

  // Decision intelligence (from Engine)
  String explanation;         // Generated by engine
  double confidenceScore;     // Calculated by engine
  List<String> recommendations; // Generated by engine

  // Metadata
  String engineVersion;       // "v2.1"
  DateTime calculatedAt;      // timestamp
}
```

## 🎯 Usage Pattern (NEW)

```dart
// 1. Calculate everything from engine
final output = SmartFeedDecisionEngine.buildSmartFeedOutput(...);

// 2. Convert to FeedResult for UIBackward compat
final feedResult = SmartFeedDebugHelper
  .buildFeedResultFromOutput(output);

// 3. Display on screen
Navigator.push(
  context,
  MaterialPageRoute(
    builder: (_) => SmartFeedDebugScreen(data: feedResult),
  ),
);
```

## 🎯 Confidence Scoring Rules

```
Base Score:           0.50

+ Recent Sampling (0-3 days):    0.15
+ Sampling (4-7 days):          0.10
+ Sampling (8-14 days):         0.05

+ FCR Factor:                   0.07
+ Tray Factor:                  0.07
+ Growth Factor:                0.06

+ Smart Phase (DOC > 30):       0.05
+ Multi-factor Consistency:     0.05

= Final Score (0.0 - 1.0)
```

## 📊 Recommendation Rules

```
FCR < 0.90     → Reduce 5-10% for 3 days
FCR < 0.95     → Slightly reduce for 2 days
FCR > 1.10     → Consider increasing feed
Tray > 1.20    → Continue current feeding
Tray < 0.80    → Check for diseases
Growth > 1.05  → Maintain feeding
Growth < 0.95  → Increase carefully
Sampling > 10d → Measure ABW today
Sampling > 7d  → Plan measurement within 2 days
DOC source     → Take fresh sampling
Low confidence → Verify manually
```
